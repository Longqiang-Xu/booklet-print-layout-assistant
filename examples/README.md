# Examples

This directory contains copyright-safe sample files for trying Booklet Print Layout Assistant.

`sample-17-pages.pdf` is generated by:

```powershell
python .\scripts\create_example_pdf.py
```

The sample has 17 reading-order pages. With the default `--max-sheets 14`, it
stays in one booklet with three blank pages added. With `--booklet-count 2`, it
splits into two smaller booklets.
